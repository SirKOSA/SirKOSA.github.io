Graduation work

For my graduation work I have made multiplayer game using Blender and Unity.

The game "Tough Week" is a local multiplayer game that supports up to 4 players. During gameplay, players can choose their own color and hat for their character, and compete in a series of challenges to win the tournament. Player with the most wins at the end of the tournament, wins

Players can join the game using keyboard or controller by pressing single button. The game lets players setup tournament with single level or let the game choose diffrent level for each round randomly. Available levels are determined by the amount of players that joined the game.

The game has levels that each have their own implementation of scoring players and some randomness. When the winner is selected the event about winner is triggered and the tournament singleton saves winners id into scoreboard
